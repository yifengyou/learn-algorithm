# Python模板

## 链表

```
class ListNode (object):
    def __init__(self, data):
        self.data = data
        self.next = None
```

## 二叉树

```
class TreeNode (object):
    def __init__(self, data, left = None, right = None):
        self.Left = None
        self.Right = None
        self.data = data
```
